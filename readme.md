# 亦幸的 Neovim 配置

## 安装 Neovim 0.7

```shell
git clone https://github.com/neovim/neovim.git
cd neovim
git checkout release-0.7
make CMAKE_BUILD_TYPE=Release
sudo make install
```

## 安装配置

```shell
git clone git@github.com:hencter/nvim.git ~/.config/nvim
```
